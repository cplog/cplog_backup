Alphavantage (https://www.alphavantage.co/documentation/)

Welcome to Part 2. This is part is about getting historical price of stocks. If you are intrested in reading data for whole industry of stock in Hong Kong, you can refer to part 1 (https://cplog.github.io/blog/stock-statistics/) .

For open source project, geting free and easy data is always the challenge. I have tried serveral resources for free stock data, and alphavantage is definitely one of the easiest and full features that save tons of time in preprocessing. Welcome to let me know if you found any tricks or down side. Here are some pros and cons from my experience:

Pro:

  1. Totally open source
  2. Ready to use indicators (e.g. rsi, sma)
  3. Different time series (e.g. weekly, monthly)
  4. Stock data all over the world
  5. Free intraday data
  6. Batch stocks enquiry

Con:

  1. Doesn’t support HK stock intraday data (maybe some other countries as well)
import pandas as pd
from toolbox import alphavantage_api
import requests as requests
from pandas.io.json import json_normalize

#You need to apply you own api key from their website which is free
#And I just stored it in toolbox

def get_stock(stock,alphavantage_api):
    df = pd.read_csv('https://www.alphavantage.co/query?function=TIME_SERIES_MONTHLY'
                     '&symbol={}'
                     '&apikey={}'
                     '&datatype=csv'.format(stock,alphavantage_api))
    return df

#e.g. 0857.hk montly result up till 2000
display(get_stock('0857.HK',alphavantage_api).head())
display(get_stock('0857.HK',alphavantage_api).tail())

Technical Indicator (e.g. RSI)

When it comes to technical indicator, seems it does not support csv output yet.
But we can still handle it by requests and treat it a json
Below shows an example of monthly rsi period=9

def get_rsi(stock,alphavantage_api):
    df = requests.get('https://www.alphavantage.co/query?function=RSI'
    '&symbol={}'
    '&interval=monthly'
    '&time_period=9'
    '&series_type=close'
    '&apikey={}'.format(stock,alphavantage_api)).json()
    return df

#Then we can also convert it to dataframe if needed
display(json_normalize(get_rsi('0857.HK',alphavantage_api)['Technical Analysis: RSI']).T.tail())

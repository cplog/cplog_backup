import pandas as pd
from requests_html import HTMLSession
import re
import time
import os
import numpy as np

curDir=os.getcwd()

Fetch a list of industry stock ticker from etnet

  1. use requests-html package to get the html contents > example of oil industry

  2. The absolute links with this https://www.etnet.com.hk/realtime/quote.php\?code=(.*) can extract the tickers

session = HTMLSession()
oil='ONG'
hotel='HNE'
#Parameter
industry=oil

r = session.get('https://www.etnet.com.hk/www/eng/stocks/industry_detail.php?nature={}&subtype=all'.format(oil))

pattern = re.compile(r"^https://www.etnet.com.hk/realtime/quote.php\?code=(.*)")
industry_list={pattern.search(i).group(1) if pattern.match(i) != None else None  for i in r.html.absolute_links}
industry_list.remove(None)

Pre-work

Before I read those files, I already downloaded statis for the related stocks in /dbstatistics/<industry name><stock no>.csv
The column name are the corresponding years. Here we take TTM (Trailing Twelve Month)

series_list=[]
for i in industry_list:
    new=pd.read_csv(curDir+'/db_statistics/{}_{}.csv'.format(industry,i),dtype=object, sep=',',index_col=[0])
    
    new_df=pd.DataFrame(new['TTM']).rename(columns={'TTM':'TTM_{}'.format(i)})
    
    series_list.append(new_df)    

# 1st stock
